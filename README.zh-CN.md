#### 介绍
`p2p`内网聊天软件。程序使用`Slint-UI`和`Rust`编写。

#### 功能
- [x] 支持自动发现节点
- [x] 支持刷新列表
- [x] 支持中英界面
- [x] 支持发送文本
- [x] 支持发送图片
- [ ] 支持发送文件

#### 如何构建？
- 安装`Rust`和`Cargo`
- 执行`make build`
- [Makefile](./Makefile)了解更多
